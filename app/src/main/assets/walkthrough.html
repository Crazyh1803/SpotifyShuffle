How to Get Your Spotify Client ID

Complete these steps once — takes about 2 minutes.

1
Open the Spotify Developer Dashboard

Go to developer.spotify.com/dashboard and log in with your Spotify account. Then tap Create app in the top-right corner.

2
Fill in the Create App form

Enter any App name and Description (for example, "True Shuffle" and "A shuffle of all my music").

In the Redirect URIs field, enter this exactly, then tap Add:

com.spotifytrueshuffle://callback

Under Which API/SDKs are you planning to use?, check Web API. Accept the terms and tap Save.

3
Copy your Client ID

After saving, you'll be taken to your app's Basic Information page. Your Client ID is the 32-character code shown at the top (highlighted below). Tap the copy icon next to it.

Note: You do NOT need the Client Secret. True Shuffle uses a secure method (PKCE) that only requires the Client ID.
4
Paste it in True Shuffle

Go back to the True Shuffle app, paste your Client ID into the field, and tap Continue. You'll only need to do this once.

Your Client ID is stored only on this device and is never sent anywhere except directly to Spotify's servers when you log in.
